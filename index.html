My endless curiosity drives me to explore how things work around me. I love discovering the beauty of all districts in Himachal Pradesh, working on exciting science projects, expressing my thoughts through paintings, and learning new coding skills.

def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    if y == 0:

    return x / y
